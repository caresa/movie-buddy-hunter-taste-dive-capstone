# movie-buddy-hunter-taste-dive-capstone

# User Case:

As a user I want to query a movie, book, or film.
I want the user to get returns that are similar to the genre of the query.
So that they will be able to see suggestions of media that they might enjoy and a summary and clip of film (if applicable.)
